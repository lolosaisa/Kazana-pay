Kazana Pay lets WooCommerce stores accept USDC payments on Base Testnet with NFT receipt minting.

== Description ==
Kazana Pay is a WordPress + WooCommerce plugin that allows merchants to receive payments in USDC on Base Testnet. After a successful payment, an NFT receipt is automatically minted for proof of purchase.

**Current features:**
- Accept USDC payments on Base Testnet
- Auto mint NFT receipts
- Works with WooCommerce checkout
- Converts KES currency to USDC
- Testnet mode for safe experimentation

**Planned features:**
- Mainnet support (Base)
- Multi-currency conversion (KES → USD → USDC)
- Merchant dashboard for payment history

== Installation ==
1. Upload the plugin folder `kazana-pay` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
3. Go to **Settings → Kazana Pay** and enter your merchant address and Base RPC.
4. Make a test purchase from your WooCommerce store.
